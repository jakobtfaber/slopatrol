# slopatrol

Slopatrol is a personal fork of Slopwatch focused on interpreting and policing
low-quality coding-agent work.

The inherited foundation observes coding-agent Sessions, Turns, Model requests,
and tool use. The fork direction is to add an explainable judgment layer on top:
session summaries, deterministic slop heuristics, policy evaluation, and
review-ready reports for humans.
